    You can prop push and prop kill all you want.
    Please do be ready to mash your undo button when something unusual starts happening.

  • Please be civil!

    Please do not attack others for having a different opinion!
    Be a nice human being and accept that others can have different opinions!

  • Please have common sense.

    Even if it isn't in the rules, if it sounds like a bad idea, then please dont do it!
    Please check with an admin if you have mixed opinions on it.

  • Clientside Lua.

    CSLua is allowed and you wont be banned for using it to cheat.
    You WILL BE BANNED if you abuse it to crash or lag the server and/or its players!

    The whole point of a Sandbox server is to have fun. Right?
    Make sure while you build and/or PvP that the other people are having fun too. :)
